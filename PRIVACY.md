# Privacy Policy

- This extension does not collect, store, or share any personal information.
- All user settings and manual block lists are stored locally in the user's browser and are never transmitted to any external server.
- If the user voluntarily configures a "Remote Subscription URL", the extension will periodically fetch that specific URL in the background to retrieve a public block list. No personal data or browsing history is sent during this request.
- The extension only operates on eksisozluk.com and its alternative domains, and does not interact with other websites except for the aforementioned user-provided URL.
