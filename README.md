<h1 align="center">uSozluk</h1>

With uSozluk, you can block users, titles, and entries, view images within entries through a gallery preview, and auto-click "continue reading" buttons. All changes are applied instantly; giving you full control over your topic lists, entry pages, user profiles, and links/images.

## Install

- [Chrome Web Store](https://chromewebstore.google.com/detail/igblepkjbdkflahkbjebobenplojogfm)
- [Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/-)
- [Firefox Add-on](https://addons.mozilla.org/en-US/firefox/addon/usozluk/)

## Features

- **Mobile Support** — Use uSozluk on all Chromium-based/Firefox-based browsers (via Quetta Browser/Firefox Mobile).
- **Block Users, Titles, and Entries** — instantly hide content you don't want to see; blocked items are removed from the DOM.
- **Remote Subscriptions** — Subscribe to any JSON blocklist URL.
- **Auto-Sync** — Automatically update all your subscriptions at 6 hours / 12 hours / 1 day / 2 days / 1 week intervals.
- **Manual & Bulk Update** — "update now" button next to each list, or a single "update all" button for all lists.
- **Quick Block Buttons** — add a one-click ✕ block button next to usernames, titles, and entries.
- **Universal Blocklist** — Community-maintained, pre-loaded blocklist with 2,700+ users (add with a single click). Powered by berkaygediz/universal-blocklist.
- **Image Gallery** — display image links inside entries as a horizontal thumbnail strip; full-screen lightbox with keyboard navigation.
- **Auto-Expand** — automatically click all "continue reading" buttons.
- **Manage Your Lists** — export/import your block lists as JSON, merge them, or clear them.
- **Real-Time Updates** — changes are applied instantly across Ekşi Sözlük.
- **Customizable Interface** — on/off toggle for each setting; accessible from the options page or popup.
- **Privacy Friendly** — runs only on Ekşi Sözlük domains; no tracking, no servers, no analytics, no ads.
